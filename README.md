# 📱 PHONE STORE - HỆ THỐNG QUẢN LÝ VÀ BÁN LẺ ĐIỆN THOẠI DI ĐỘNG

Dự án website thương mại điện tử chuyên kinh doanh các dòng điện thoại thông minh, tích hợp trang quản trị (Admin Dashboard) mạnh mẽ và hệ thống hiển thị thông số kỹ thuật chi tiết cho khách hàng.

## 🛠️ CÔNG NGHỆ SỬ DỤNG

Hệ thống được phát triển trên nền tảng Java hiện đại kết hợp với các framework tối ưu hiệu năng:

| Thành phần          | Công nghệ / Thư viện sử dụng                                     |
| :------------------ | :--------------------------------------------------------------- |
| **Backend Core**    | Java, Spring Boot (Spring MVC, Spring Data JPA, Spring Security) |
| **Database**        | MySQL                                                            |
| **Template Engine** | Thymeleaf                                                        |
| **Frontend UI**     | Bootstrap 5, HTML5, CSS3, JavaScript                             |

## 🚀 CÁC CHỨC NĂNG CHÍNH CỦA HỆ THỐNG

### 👥 1. Phân hệ Khách hàng (Client Website)

- **Đăng ký, đăng nhập:** Đăng ký tài khoản mới vào hệ thống, đăng nhập sau khi đăng ký thành công.
- **Xem danh sách sản phẩm:** Bộ lọc thông minh theo Hãng sản xuất (Danh mục) và sắp xếp theo giá cả.
- **Xem Chi tiết sản phẩm:** \* Hiển thị mô tả quảng cáo, giá bán, tình trạng kho hàng.
- **Giỏ hàng & Thanh toán:** Thêm sản phẩm vào giỏ, cập nhật số lượng, tính toán tổng tiền và tạo đơn đặt hàng trực tuyến.
- **Đánh giá sản phẩm:** Đánh giá sản phẩm sau khi đơn hàng hoàn thành.

### 💼 2. Phân hệ Quản trị (Admin Dashboard)

- **Quản lý danh mục:** Thêm, sửa, xóa các hãng điện thoại (Apple, Samsung, Xiaomi...).
- **Quản lý sản phẩm:** Thêm, sửa, xóa thông tin điện thoại
- **Quản lý đơn hàng & Kho hàng:** Theo dõi trạng thái đơn hàng của khách.
